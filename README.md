# How To Install DM

linux:

git clone https://github.com/mostafa-vn/DownloadManager.git && tar -jz && cd DownloadManager && chmod +x dm && mv dm /usr/local/bin


windows:

git clone https://github.com/mostafa-vn/DownloadManager.git && tar -jz && cd DownloadManager && chmod +x dm && mv dm /usr/local/bin
